**API** - Application Programming Interface
**REST** - Representational State Transfer

**API** organizes resources into a unique set of URI's
**URI** - Uniform Resource Identifiers

**POST** - Create (request)
**GET** - Read
**PUT** - Update
**DELETE** - Delete

**JSON** - JavaScript Object Notation
- Used as a way to read the data 

**Responses**:
- **200** - Success
- **400** - Something wrong with request
- **500** - Something wrong with server

**Database** <--> **Python BackEnd** <---> **API Request** <--> **JavaScript FrontEnd**
